Aplikasi ini adalah tugas Mata Kuliah Pemrograman Mobile sederhana menggunakan Kotlin. Fitur utama meliputi:
1. Input Nama (TextInput).
2. Tombol "Sapa" untuk menampilkan salam.
3. Teks dinamis (TextView) "Hello {nama}!".
4. Switch Toggle untuk mengubah tema Mode Gelap/Terang secara instan.

Cara Menjalankan
1. Download atau Clone repository ini.
2. Buka aplikasi Android Studio.
3. Pilih Open dan arahkan ke folder proyek ini.
4. Tunggu proses Gradle Sync selesai.
5. Klik tombol Run (ikon segitiga hijau) untuk menjalankan di Emulator atau HP fisik.
